# tree-cli
A simple cross-platform alternative to the unix `tree` command.

### 🚧👷‍♂️ Construction in progress
## Example 
```shell
❯ tree-cli
.
├── Cargo.lock
├── Cargo.toml
├── README.md
├── config
│   └── config.toml
├── rustfmt.toml
└── src
    ├── core.rs
    ├── file_iterator.rs
    ├── filter.rs
    ├── main.rs
    └── symbol.rs

3 directories, 11 files
```
